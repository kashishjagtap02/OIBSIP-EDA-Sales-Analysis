# Retail Sales Data Analysis (EDA)

## Internship Project

This project was completed as part of the Oasis Infobyte Data Analytics Internship Program.

## Project Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing patterns, sales trends, and product performance. The analysis helps derive actionable business insights for decision-making.

## Tools and Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Information

The dataset contains 1000 retail sales transactions with the following attributes:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Analysis Performed

### Data Cleaning

* Checked for missing values
* Checked for duplicate records
* Verified data types
* Converted date column into datetime format

### Exploratory Data Analysis

* Gender-wise Sales Analysis
* Product Category Analysis
* Monthly Sales Trend Analysis
* Age Group Analysis
* Correlation Heatmap

## Key Findings

* Female customers generated slightly higher sales revenue than male customers.
* Electronics was the highest-performing product category.
* Customers aged 46+ contributed the highest revenue.
* Sales varied across different months, indicating seasonal trends.
* Price per Unit showed a strong positive correlation with Total Amount.
